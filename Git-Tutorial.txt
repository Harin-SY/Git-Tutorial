git --version
깃 버전 옵션 확인

git config --global -> 컴퓨터 전체에서 동일한 옵션을 사용하게 해주는 명령어

<환경설정>
git config --global user.name 깃허브 이름

git config --global user.mail 깃허브 이메일 주소

<동기화>
cd ->폴더 이동 명령어

cd 폴더 주소

git clone -> 다운로드

git clone 깃 저장소 주소

<변경사항 업로드>
cd Python-basic

git add -> 파일 업로드 명령어

git add 2-ProgrammingInPython.txt 


업로드 시점 스냅샷 찍기 - 문제 발생시 복구 작업 때 사용함
git commit -m "Add Text File [Git-Tutorial.txt]"


git push -> 원격 저장소에 적용시키는 명령어
